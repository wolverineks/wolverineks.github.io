Ruby vs Javascript: Iterating Over an Array

February 14, 2016

Both Javascript and Ruby have a built in mechanism for iterating over arrays.

In ruby, you can initialize a variable simply by typing the name of the variable and then assigning the array to that variable.

Then you use the .each method, use the do..end syntax, and include the code that you wish to execute for each element of the array.

In javascript, you can iterate over an array by initializing a variable with the keyword 'var', typing the name of the variable you wish to use, and assigning the array to that variable.

Then, you use the for syntax to iterate over the array. The 'i' is used as an index counter. It is incremented with the 'i++' syntax. Inside the { } you write whateve rcode you wish to execute for each element of the array.
